🚀 Expo

Gradle cache for Android builds

ExpoでAndroidビルド時のGradleキャッシュが有効になり、ビルド時間が大幅に短縮されます。キャッシュは自動的に管理され、依存関係の変更時に無効化されるため、開発効率が向上します。既存のプロジェクトで即座に効果が期待できます。

🐦 Flutter

Saying goodbye to CocoaPods: Swift Package Manager is soon the default in Flutter!

Flutter 3.44以降、iOSおよびmacOSアプリの依存関係管理がCocoaPodsからSwift Package Manager(SwiftPM)にデフォルト変更されます。CocoaPodsは2026年12月2日にレジストリが読み取り専用になるため、今後の依存関係更新とSwiftパッケージエコシステムへのアクセスを確保するにはSwiftPMへの移行が必要です。既存アプリは引き続き動作しますが、新機能やセキュリティアップデートを受け取るには、プラグインやプロジェクトをSwiftPMに対応させる必要があります。
